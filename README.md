# Stack Overflow Survey Analizi

Bu proje, Stack Overflow 2025 Developer Survey verilerini analiz eden bir Streamlit uygulamasıdır. 

## Özellikler
- Veri seti genel bakışı ve istatistikleri
- Ham veri inceleme
- Sayısal değişken analizi (PCA, Standardizasyon)
- Kategorik değişken analizi (Etkileşimli Pie Chart)

## Kurulum

Gerekli kütüphaneleri yüklemek için:

```bash
pip install -r requirements.txt
```

## Çalıştırma

Uygulamayı başlatmak için:

```bash
streamlit run app.py
```
